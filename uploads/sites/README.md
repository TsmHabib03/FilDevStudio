# Site Uploads

This folder stores uploaded images for client sites.

## Structure
Each site has its own subfolder:
```
sites/
├── 1/
│   ├── logo_xxxxx.png
│   ├── hero_xxxxx.jpg
│   └── gallery_xxxxx.jpg
├── 2/
│   └── ...
```

## Image Types
- `logo` - Site logo (PNG with transparency recommended)
- `hero` - Hero/banner image (1920x800px or larger recommended)
- `gallery` - Gallery/product images
- `product` - Product-specific images
- `other` - Other images

## Requirements
- Max file size: 5MB
- Allowed formats: JPEG, PNG, GIF, WebP
- Folder must be writable by web server
